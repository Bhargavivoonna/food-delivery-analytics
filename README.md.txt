# Food Delivery Analytics Dashboard

This project analyzes food delivery data using **MySQL, MongoDB, Python, and Tableau**.

## Features
- MySQL: Orders, Users, Restaurants database
- MongoDB: Customer Reviews
- Python (Pandas, Matplotlib): Analysis & Visualization
- Tableau: Interactive Dashboard

## Sample Insights
- City-wise demand
- Popular cuisines
- Average delivery time
- Revenue by restaurant
